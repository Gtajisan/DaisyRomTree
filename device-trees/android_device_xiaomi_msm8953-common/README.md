# Common Device Tree for Xiaomi MSM8953 Devices

This repository contains a common device tree for Xiaomi devices based on the Qualcomm MSM8953 platform.

## Supported Devices

- Xiaomi Mi A2 Lite (daisy)
- Xiaomi Mi A1 (tissot)
