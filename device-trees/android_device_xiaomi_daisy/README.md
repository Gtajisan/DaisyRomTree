# Device Tree for Xiaomi Mi A2 Lite (daisy)

The Xiaomi Mi A2 Lite (codenamed _"daisy"_) is a mid-range smartphone from Xiaomi.

It was announced and released in July 2018.

## Device specifications

| Device                  | Xiaomi Mi A2 Lite                               |
| ----------------------- | :---------------------------------------------- |
| SoC                     | Qualcomm MSM8953 Snapdragon 625                 |
| CPU                     | 8x Cortex-A53 @ 2.0GHz                          |
| GPU                     | Adreno 506                                      |
| Memory                  | 3GB / 4GB RAM (LPDDR3)                          |
| Shipped Android Version | 8.1.0                                           |
| Storage                 | 32GB / 64GB eMMC 5.1                            |
| Battery                 | Non-removable Li-Po 4000 mAh                    |
| Dimensions              | 149.3 x 71.7 x 8.8 mm                           |
| Display                 | 2280 x 1080 (19:9), 5.84 inch                   |
| Rear Camera 1           | 12 MP, f/2.2, 1.25μm, PDAF                      |
| Rear Camera 2           | 5 MP, f/2.2, 1.12μm, depth sensor               |
| Front Camera            | 5 MP, f/2.0, 1.12μm                             |
